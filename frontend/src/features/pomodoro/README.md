# Pomodoro Feature

The Pomodoro feature helps users stay focused by dividing study time into work sessions (typically 25 minutes) separated by short breaks (5 minutes).  
Key functionalities include:
- Pomodoro timer with Start/Stop/Reset controls
- Alarm sound at the end of each session
- Customizable session and break durations
- Accurate countdown even when switching browser tabs

This folder contains all frontend source code for the Pomodoro Timer.
